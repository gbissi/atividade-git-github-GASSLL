# Diário de Commits

Commit 1 — Criação dos arquivos e do repositório

Inicialização do repositório com git init -b main. Criação da estrutura
de pastas do projeto: docs/ para documentação e src/ para o código-fonte.

 Commit 2 — Desenvolvimento da calculadora

Criação de src/main.py com toda a lógica da calculadora de médias:
 Leitura de 4 notas (N1, N2, N3, N4)
 Validação do intervalo (0,0 a 10,0)
 Cálculo da média aritmética simples
 Exibição da situação: Aprovado (>= 7,0), Recuperação (>= 5,0) ou Reprovado (< 5,0)
 Tratamento de entradas inválidas com mensagem de erro
